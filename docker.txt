
What is Docker?


Docker is a software that package, produce , and run an image called Docker Image with the information 
or requirement that has been provided during the build up stage by the user.


Docker file: This file consist of the requirement needed to build an image and run it seeemlesssly. 
             I.e; building a Docker Image for python file called CDE_2, then Docker File must consist the path 
             to the Python file, Python version and other dependecies that are require to run the image 
             after it has been built.


Docker Image: Is the image that was built (CDE_2) with the file that consist the requirement we proived in the Docker File during build up.


Docker Container: Is the activated version of a Docker Image, which means we can activate or run more container from a our Docker (CDE_2).
